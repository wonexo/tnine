    Price Calculation

    Before you begin

    To integrate our API successfully, you need an API key to authenticate your request. You can find your keys on the by following the steps on the authentication page.

    The pricing endpoint is the first API you will need to consume. This endpoint requires you to provide specific details of your package like the item weight and distance of delivery. once the JSON request below is sent successfully, you'll get a response that provides you with our pricing and delivery options to choose from.

    Request Body

    Make sure your have your valid authorization key required to succeffuly perfom this request.

    Sample endpoint:

    https://api-shipping-dev-kangaroo.azurewebsites.net/api/pricing-calculator/calculate

    {
    "stateId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "itemCategoryId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "weight": 0,
    "distance": 0,
    "deliveryDuration": 0
}

    curl -X POST "https://api-shipping-dev-kangaroo.azurewebsites.net/api/pricing-calculator/calculate" 
-H  "accept: */*" 
-H  "Content-Type: application/json-patch+json"
-d "{\"stateId\":\"3fa85f64-5717-4562-b3fc-2c963f66afa6\",\"itemCategoryId\":\"3fa85f64-5717-4562-b3fc-2c963f66afa6\",\"weight\":0,\"distance\":0,\"deliveryDuration\":0}"

    {
  "":""
}

    Request Parameters

    Parameter Type Description
    stateId string *required
    itemCategoryId string *required The item category identifier
    weight integer *required The weight of item
    distance integer *required The distance of item to from picup to destination
    deliveryDuration integer *required The evaluated duration or time it will take to get the item delivered to a customer
