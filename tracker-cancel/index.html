    Tracker Cancellation Reason

    This section provides all the endpoints and methods that can enable you get more information regarding reasons for tracking cancellation.

    Create Tracking Cancellation Reason

    Use this method to create a tracking cancellation reason.

    Make sure your have your valid authorization key required to succeffuly perfom this request.

    Sample endpoint:

    https://api-shipping-dev-kangaroo.azurewebsites.net/api/tracker-cancellation-reason/Create

    {
    "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "description": "string"
}

    curl -X POST "https://api-shipping-dev-kangaroo.azurewebsites.net/api/tracker-cancellation-reason/Create"
 -H  "accept: */*" 
 -H  "Content-Type: application/json-patch+json" -d "{\"id\":\"3fa85f64-5717-4562-b3fc-2c963f66afa6\",\"description\":\"string\"}"

    {
    "data": {
        "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
        "description": "string"
    },
    "success": true
}

    Request Parameters

    Parameter Type Description
    id string *required Item tracker ID
    description string *required Information containing reasons for canceling a tracker
